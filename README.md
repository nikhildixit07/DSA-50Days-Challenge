## üìÑ File Naming Convention

Each solution file in this repository follows the naming format:

```DayDD QP LeetCode ProblemID.ext```

### Breakdown of the format:

| Component | Meaning |
|----------|----------|
| **DayDD** | Represents the challenge day ‚Üí (Day01, Day14, Day37, ...) |
| **QP** | Question number solved on that day ‚Üí (Q1, Q2, Q3 if multiple) |
| **LeetCode ProblemID** | The exact LeetCode problem ID used ‚Üí (LeetCode 258, LeetCode 876, etc.) |
| **ext** | File format ‚Üí (.py, .cpp, .java, etc.) |

### üìù Example Files Explained

| Filename | Meaning |
|----------|----------|
| `Day01 Q1 LeetCode 258.py` | Day 01 ‚Üí 1st Question ‚Üí LeetCode #258 ‚Üí Python solution |
| `Day01 Q2 LeetCode 876.py` | Day 01 ‚Üí 2nd Problem ‚Üí LeetCode #876 ‚Üí Python |
